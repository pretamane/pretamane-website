Place your CV PDF file here with the filename: `cv.pdf`

- **Path:** `assets/docs/cv.pdf`
- **URL:** `/assets/docs/cv.pdf`

2. Rename it to `cv.pdf`
3. The Download CV button in the navigation will automatically work
